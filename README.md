# Email Signature — Agent Marketing Desk

Self-hosted email signature for Greg St. Fort / Agent Marketing Desk.

## Setup

1. Open `signature.html` in a browser
2. Select all (Ctrl+A / Cmd+A) and copy (Ctrl+C / Cmd+C)
3. Paste into your email client's signature settings

### Gmail
Settings → See all settings → General → Signature → Paste

### Outlook (Desktop)
File → Options → Mail → Signatures → Paste

### Outlook (Web)
Settings → Mail → Compose and reply → Email signature → Paste

## Editing

- Update `signature.html` to change text, links, or layout
- Replace images in `/images/` to update logos or icons
- Images are served from GitHub raw URLs — changes are live after push
